docstruct by cfournie

A tool to create Document Structure trees from XHTML websites.

Websites on the internet represent a vast collection of human generated textual information which could be harvested for a variety of NLP purposes. Unfortunately, websites are often of a highly graphical, or interactive nature, and do not have enough textual content to lend themselves easily to conversion into a purely textual format. This is a problem faced by screen readers for the blind, in addition to researchers trying to glean coherent text from the web. This project describes methods for selecting an appropriately parseable website, and a tool to parse it into an XML encoded Document Structure tree, and in future work, converted into Rhetorical Structure trees.

Dependencies

  • Python 2.6.x
  • python-lxml 2.0+
  • libxml2
  • libxslt
  • python-igraph 0.5+;
    • igraph;
    • cairo (easily installed as a part of GTK);
    • python-cairo (a version compatible with python-igraph).
